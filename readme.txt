It lets user adds start and end date of a page for publishing and after end date has been passed, page's status is automatic marked as draft.

== Description ==

This plugin is very useful for those who want to display their pages (on front end) from some specific date (start date) to some specific date (end date). It creates a meta box in Edit Page where Admin user can add start & end date/time. After End date has been passed, the page will expire and is saved as 'draft'. Very fast, easy to use plugin for both single & multi site installation. It uses GMT time format.


== Installation ==

1. Download the plugin and extract the files
2. Upload "page-start-expire" folder to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add Start and End Date from Add/Edit Page to see the results.
